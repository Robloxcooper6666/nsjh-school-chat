# 🚀 南勢國中即時看板 (NSJH LIVE Board)

這是一個專為南勢國中打造的即時互動空間。無論是校園生活雜談、課業請教，還是不敢說出口的告白，都能在這裡即時傳遞給全校。

---

## ✨ 功能亮點

* **全校同步**：訊息發布後，所有打開網頁的人都會在「一秒內」同時看到，完全不需要手動重新整理頁面。
* **在線監測**：右上角即時顯示目前全校有多少同學正盯著看板，增加社群臨場感。
* **話題分類**：內建四大分流，讓資訊不混亂：
    * `🌿 生活`：日常大小事分享。
    * `💖 告白`：不敢當面說的話。
    * `📚 課業`：功課求救與討論。
    * `🤫 秘密`：不能說的秘密。
* **智慧記憶**：瀏覽器會自動記住你的班級和姓名，下次進入頁面發文超省時。
* **隱藏管理室**：內置管理員模式，可監控連線人數與 ClientID（觸發方式：點擊標題 5 次）。

---

## 💡 它是怎麼運行的？（運作原理）

這款看板之所以能做到「即時」，是因為背後運用了以下原理：

### 1. 即時雙向連線 (WebSocket)
一般的網頁像「寫信」，你主動詢問，伺服器才會給你回信。
而本看板像「打電話」，當你打開網頁時，瀏覽器會與雲端建立一條**專屬且透明的管道**。只要管道不斷開，訊息就能在裡面雙向自由流動，達到零延遲的效果。

### 2. 廣播發佈機制 (Pub/Sub)
當你發送一則訊息時，系統就像一個**廣播電台**。它會接收你的訊號，然後瞬間把這份內容「廣播」給所有正連接在同一個頻率（頻道）上的同學設備上。

### 3. 本地儲存與緩存 (LocalStorage)
你的對話紀錄和個人設定會暫存在你設備瀏覽器的**專屬保險箱**裡。就算關掉網頁或網路斷開，之前的紀錄也會從保險箱讀取出來，確保體驗不中斷。

## 🛡️ 使用守則

* **尊重他人**：這是一個自由的空間，請不要發布人身攻擊或不當內容。
* **保護隱私**：分享心情之餘，也請保護自己與他人的隱私資訊。
